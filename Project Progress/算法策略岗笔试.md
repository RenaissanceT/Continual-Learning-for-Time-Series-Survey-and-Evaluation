#### 第一道题

牛牛给出了一个长度为 n 有关于未知量的式子，他想知道 x 的一次项系数是多少？

牛牛以字符串的形式给出这个式子，这个式子会由若干个 (x - d) 和 (x + d）的括号式组成，括号式之间是乘积关系。其中 d(1 ≤ d ≤ 9) 是具有数字意义的宇符。

具体格式可见样例。

> **输入描述**
> 
> 一行一个字符串，长度为 n(5 ≤ n ≤ 100000）


> **输出描述**
> 
> 一行整数，表示未知量 x 的一次项系数，对 10007 取模


 **示例 1**

> **输入**
> 
> (x - 1)(x + 5)


> **输出**
> 
> 4



> **说明**
> 
> 展开后得到：x*^2 + 4*x - 5
> 
> 显然 x 的一次项系数是 4

#### 第三道题

圣诞节快到了，圣诞老人按照小朋友的愿望准备了 n 个礼物，其中第 i 个礼物的长度为 a<sub>i</sub>，宽度为 b<sub>i</sub>。现在圣诞老人需要一些彩纸来将礼物包装起来。彩纸店恰好提供了 n 种彩纸，每一种彩纸的长宽比例是固定的，但是彩纸的大小可以自选。

具体来说，第 i 张彩纸的长度为 c<sub>i</sub> * k，宽度为 d<sub>i</sub> * k, 其中 k 可以自己决定。当且仅当某一张彩纸的长度大于等于礼物的长度，并且宽度大于等于礼物的宽度，这张彩纸才可以将这个礼物包装起来。

一张彩纸仅可用于包装一个礼物(如果彩纸的长宽有多出来的部分，则舍弃）。

现在圣诞老人想知道，如果他想用这 n 张彩纸包装 n 个礼物，k 的最小值是多少？

> **输入描述**
>
> 第一行输入一个正整数 n
>
> 接下来有 n 行，每行包含四个正整数，分别表示 a<sub>i</sub>, b<sub>i</sub>, c<sub>i</sub>, d<sub>i</sub>
>
> 1 ≤ n ≤ 50000,1 ≤ a,b,c,d ≤ 100000
>
> 注意，本题不保证 a<sub>i</sub> ＞ b<sub>i</sub>，c<sub>i</sub> ＞ d<sub>i</sub>，但是信封不可旋转，即你可以认为信封的 a<sub>i</sub> 仅对应彩纸的 c<sub>i</sub>，信封的 b<sub>i</sub> 这条边仅对应彩纸的 d<sub>i</sub>。


> **输出描述**
>
> 输出一个正整数，表示最小的 k


 **示例 1**
 
> **输入**
> 
> 2
>
> 1 1 1 2
>
> 2 4 1 1 


> **输出**
> 
> 2


> **说明**
>
> k = 2时，两张彩纸的长宽分别为 2 * 4, 2 * 2，用第一张彩纸包装第二个礼物，用第二张彩纸包装第一个礼物，这样两个彩纸都满足 “长宽均大于等于对应礼物”，所有礼物都被包装。
>
> 假如 k = 1，可以用第二张彩纸包装第一个礼物。但是第二个礼物就没有彩纸可以包装了，因而答案最小为 2。
